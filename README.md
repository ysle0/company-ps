# Company PS

회사 알고리즘 문제풀이 연습 저장소입니다.

## 참여자

이윤상
최희지

## 목표

프로그래머스 1,2 다 풀기 + 백준 골드 상위까지 노려보기.

## 구조

```text
dates/{년도}/week{주차}/{날짜}-{문제명}/
  ├── desc.md          # 문제 링크 및 정보
  └── {이름}.{확장자}   # 개인 풀이
```

## 실행 방법

**C++:**

```bash
clang++ -o FILENAME FILENAME.cpp
./FILENAME
```

**C#:**

```bash
dotnet run FILENAME.cs
```
